Русскоязычные комментарии:

    Класс progressBar создается и управляет индикатором загрузки.
    Этот класс содержит конструктор класса и 4 метода getValue, вращение, скрыть, setValue.
    Механику индикатора загрузки я постарался реализовать на подобом того, как работают большинство каруселей,
    то есть создаётся пустой блок, для него прописывается id, по которому класс создаетёт обновления.

    Конструктор класса конструктор (data = {id: строка, ширина: число, скорость: строка}) - в конструктор класса
    передаётся объект, который содержит:
    id (идентификатор блока, который будет являтся индикатором загрузки), ширина (ширина кольца индикатора),
    скорость ([мс] время заполнения индикатора загрузки). Конструктор класса набора начальных значений переменных
    класса, которые будут использоваться при его загрузке. Было принято решение поместить в конструктор класса
    формирование DOM дерева нашего индикатора.

    rotation (bool) - данный метод запускает анимацию загрузки индикаторов загрузки. Входным параметром является булевая
    переменная. Анимация реализованна при помощи css анимации и активируется / диактивируется добавлением / удалением класса
    "ProgressBar__circle_rotation"

    hide (bool) - данный метод управляет видимостью индикатора загрузки. Входным параметром является булевая
    переменная. Управление видимостью реализуется средствами css и активируется / диактивируется добавлением / удалением класса
    "ProgressBar_hidden"

    getValue (строка) - данный метод получает значения индикаторов загрузки проверяет его и
    передаёт это значение в метод setValue.

    setValue () - данный метод управляет степенью заполнения индикатора. Происходит пересчёт числа от 0 до 100 в длину дуги
    индикатора и запуск анимации заполнения. В данном проекте используется SVG анимация.
